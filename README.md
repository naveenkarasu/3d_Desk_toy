## Overview
This is a beginner Unity project designed to explore the fundamentals of the engine. The goal is to create an interactive "toy" object that rotates, changes color on click, and can be reset via a UI.

## Features
- **Interactive Object:** A 3D cube that rotates automatically.
- **Input Handling:** Click the object to change its color.
- **UI Controls:** Reset orientation and randomize colors using on-screen buttons.
- **Prefab System:** Spawns multiple instances of the toy in a grid.

## Controls
- **Mouse Click:** Change the color of the toy.
- **UI Buttons:** Use the panel on the screen to Reset or Randomize.

## Technologies
- Unity 2022.3 LTS (or your version)
- C# Scripting
